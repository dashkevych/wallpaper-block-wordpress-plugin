Simple block that allows to create interactive and animated falling snow effect.

== Description ==

In a world where every website is striving for attention, imagine if yours could tell a story, not just with words but with the atmosphere.

The plugin provides the "Holiday Snow" WordPress block to create interactive and animated falling snow effect.

Think about the last time you saw snow falling.

The quiet, the calm, the way it made you feel. That's what the "Holiday Snow" block brings to your website.

It's not just about adding snowflakes; it's about invoking the spirit of the season, creating a mood that lingers and memories that stick.

Because when they remember the snow, they'll remember you.

== Installation ==

1. Choose one of the following options:
	* Go to Plugins → Add New in your WordPress dashboard, search for "Wallpaper Block", and click "Install".
	* Download the Wallpaper Block plugin from WordPress.org, ensure the folder is zipped, and then upload it via Plugins → Add New → Upload.
    * Open the block inserter within the Block Editor, search for "icon", and the plugin should appear, allowing you to install it directly. Skip steps 2 and 3.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Search for the block within the Block Editor and start using it.


== Frequently Asked Questions ==

= What is Wallpaper Block? =

The Wallpaper Block allows to create interactive and animated section.

The block supports Align options: None, Wide width, Full width, etc.

Also, you can customize height of the block and its border styles.

The Wallpaper Block serves as a container for the main section design.

Currently, plugin supports only one design: Holiday Snow.

= What is Holiday Snow? =

The Holiday Snow Block provides interactive and animated falling snow effect.

This block allows to customize:

1. snow color
2. background overlay
3. background image
4. content position

The Holiday Snow Block can only be used within the Wallpaper Block.
